# GitHub Showcase Pack

## Repository Description (Short)

Student Performance & Grade Predictor using Scikit-learn and Streamlit with modular ML pipeline, batch CSV predictions, and clean portfolio-ready structure.

## GitHub About (Long)

This project predicts final student scores and grade bands (A/B/C/D) using attendance, assignment performance, and internal exam marks.

It is intentionally built with a clean software engineering structure:
- modular source code separation (training, prediction, model IO, UI),
- saved model loading via pickle,
- Streamlit interface for both single and batch prediction,
- basic testing and deployment-ready documentation.

The repository is suitable for:
- academic submission,
- beginner-to-intermediate ML portfolio showcase,
- interview discussion around end-to-end ML app packaging.
